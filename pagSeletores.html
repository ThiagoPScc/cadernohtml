Seletores

seletor é como chamamos os elementos no css ex: .p color:red /no caso 'p' é o seletor

* é o seletor universal, ou seja, todos os seletores em um só

seletor de texto: h1,h2,h3...

seletor de classe: quando criamos uma classe com css.

podemos usar tambem assim p.verde [color:red]

aqui seleciona todos os paragrafos que tem a classe verde,limitando os objetos que podem pegar esta classe para apenas paragrafos

seletor por id: #algo[] o que ja usamos anteriormente

seletor de filho: li>a[]

aqui o que acontece é, todos os links que estivirem dentro de um li, teram essa formatação

funciona para qualquer classe dentro de outra, seja por sintaxe ou divs, porem só funciona para elementos diretamente ligados, sendo assim não é possivel colocar um li-span-a, precisa ser direto:li-a

    seletor de descendente: p a [], igual o anterior porem agora não precisa ser diretamente ligado

    seletor adjacente ou irmão: h1+p, ele ira adcionar a classe ao proximo p grudado a um h1, somente o posterior

    seletor irmão geral: igual ao anterior, porem não é necessario ser adjacente
